---

## 5. 角色记忆

```kotlin
data class CharacterMemoryBank(
    val shortTerm: List<Memory>,
    val episodic: List<Memory>,
    val semantic: List<Memory>,
    val emotional: List<Memory>,
    val relationship: List<Memory>,
    val blocked: List<BlockedMemory>
)
```

### 写入规则

1. 只有 visible 内容能进入记忆。
2. 只有在场或被告知的角色能记住。
3. world truth 不得泄入 Actor memory。
4. 低重要性内容只进短期摘要。
5. blocked memory 只给 Actor 症状，不给隐藏原因。

## 6. 世界事实

## 9. 路线与 Arc

```kotlin
data class ArcState(
    val id: ArcId,
    val type: ArcType,
    val stage: String,
    val tags: Set<String>,
    val locked: Boolean,
    val priority: Int
)

enum class ArcType {
    MAIN, CHARACTER, SIDE, ENDING
}
```

Galgame 路线规则：

1. 主线永远存在。
2. 角色线可并行暧昧。
3. route_locked 后其他角色线降权或冻结。
4. 结局必须由结构化条件触发。
